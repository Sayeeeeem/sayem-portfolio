# Portfolio Project

This project is set up so you can update most portfolio content from one file.

## Edit Content In One Place

Update this file:

- `src/config/portfolio.ts`

From there you can change:

- Navbar links and brand text
- Hero text and CV link
- About traits
- Clubs and volunteering entries
- Certifications and awards
- Ventures
- Contact links and footer text

## Run

- Dev mode: `npm run dev`
- Static build: `npm run static:build`
- Build + open dist: `npm run static:ready`
